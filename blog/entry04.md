# Entry 4
##### 2/13/26

## Context
I experimented with different coding tools and spend time learning how they work by tinkering with them and watching videos. The tool that caught my attention the most was A-Frame. It stood out to me because of how it can create 3d objects. I believe this tool will help me create how my future devices would look like. with A-frame, i can code and visually design these devices for my website so others can see what i imagine.

## Sources
I chose A-Frame as it makes it possible to build 3D environments ans objects directly in a browser. I found that it was interesing as you could also move around zooming closer or further away from the shapes. To learn how to use A-Frame i use A-Frame school and videos on youtube to understand it better. I also practiced on my IDE using the starter code they provided, i changed the color,position and rotation of different objects to see how they changed. 

Starter code 
``` html 
  <a-scene>
      <a-box position="-1 0.5 -3" rotation="0 45 0" color="#4CC3D9"></a-box>
      <a-sphere position="0 1.25 -5" radius="1.25" color="#EF2D5E"></a-sphere>
      <a-cylinder position="1 0.75 -3" radius="0.5" height="1.5" color="#FFC65D"></a-cylinder>
      <a-plane position="0 0 -4" rotation="-90 0 0" width="4" height="4" color="#7BC8A4"></a-plane>
      <a-sky color="#ECECEC"></a-sky>
    </a-scene>
```
After
``` html
  <a-scene>
      <a-box position="-1 0.5 -3" rotation="1 1.5 -4" rotation="0 45 0" color="#4CC3D9"></a-box>
      <a-sphere position="0 1.25 -5" radius="1.25" color="#EF2D5E"></a-sphere>
      <a-cylinder position="0 1.75 -2" radius="0.5" height="2.5" color="#FFC65D"></a-cylinder>
      <a-plane position="0 0 -4" rotation="-50 10 0" width="3" height="3" color="#7BC8A4"></a-plane>
      <a-sky color="27F2F5"></a-sky>
    </a-scene>
```
This code now is change, the background is a cyan color now. The scene includes a box,sphere,cyliner, and a plane that acts as the ground. Each object now has new color,rotation,size , position or radius now. 

## Skils 
one skill i developed during this process was learning a new skill by myself. I had to explore examples,watch tutorials and test code on my own to understand how A-Frame works. By doing this, i became more confident on exploring and experimenting with learning new code and using new tools.

A skill i had improved was problem solving. While experimenting with the code, i faced challenfes when trying to adjust shapes,colors and positions to make the objects look the way i wanted. 

I also improve my creativty while with A-Frame. I had to think about what colors,shapes, and positions would look best in the scene.  Such as making new shapes with basic ones, i would have to combine the diferent shapes to the one i wanted.
